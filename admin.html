Approval Process
How it works: FLOW uses a simple email allowlist in login.html. When someone signs in with Google, their email is checked against the APPROVED_USERS object. If their email is there, they get in. If not, they see a "Pending" screen.

To approve a user:

1. Find the user in the Pending tab
2. Click ✓ Approve — this copies the code snippet
3. Open login.html in a text editor
4. Find the APPROVED_USERS object and paste the line
5. Redeploy to Netlify
6. User can sign in immediately

The APPROVED_USERS object looks like this:

var APPROVED_USERS = {
  'markus@email.com': { role: 'admin', name: 'Markus' },
  'ryan@email.com': { role: 'user', name: 'Ryan' },
  // Add more users here
};
